# Landing Page Templates Portfolio

A collection of beautiful, responsive website templates including SaaS landing pages, portfolios, restaurant sites, and more. Built with HTML, CSS, JavaScript, and Tailwind CSS.

**Author**: Vara Prasad  
**GitHub**: https://github.com/Prasadchowdar

## Features
* Fully responsive designs
* SEO optimized (semantic HTML tags)
* Tailwind CSS with `tw-` prefix
* Easy customization
* Framework independent (HTML/CSS/JS)

## Installation and Setup

### Prerequisites
- Node.js installed

### Setup
1. Install dependencies:
```bash
npm install
```

2. Start Tailwind (development):
```bash
npm run start:tailwind
```

3. Build Tailwind (production):
```bash
npm run build:tailwind
```

## Customization
1. Change text content directly in HTML files
2. Modify colors in `tailwind.config.js` (primary/secondary colors)
3. Update styles in `index.css` for buttons, inputs, etc.
